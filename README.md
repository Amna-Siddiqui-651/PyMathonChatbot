# PyMathonChatbot

A Python-based math chatbot that performs mathematical calculations and interacts with users through both manual and natural input commands.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Floor Division
- Power Calculations
- Modulus
- Factorial
- Square Root
- Percentage Calculator
- Even/Odd Checker
- Tables Generator
- Calculation History
- Clear History
- Natural Input Support

## Example Commands

add 5 10 15
sub 20 5 2
multiply 2 3 4
history
clear history
